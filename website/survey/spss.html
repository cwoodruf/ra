SPSS Howto

SPSS is a statistics software package used in social research. Use SPSS to summarize interview data, test hypotheses and check data quality.

Save data to a file

Use the CSV or TAB links in Research Assistant to save section interview data in an SPSS readable format.

Import data into SPSS

These instructions have been tested with SPSS version 22.

For CSV data use the "File > Read Text Data..." dialog to import text data. This can also be used for TAB data.

Alternatively, for TAB files, click on the "VARIABLE LABELS for TAB data" link to generate a script to automatically import the TAB file and create descriptive variable labels. Open a new SPSS sytax window with the "File > New > Syntax" menu option. Paste the VARIABLE LABELS output into the sytax window.

Edit the SPSS GET command to point to the TAB data file saved earlier. Use the "Run > All" menu option in the syntax window to run all commands. To run a single command click on the command and either click the green arrow in the toolbar or use the "Run > Selection" to run each command in turn.

To set variable labels for CSV files copy the VARIABLE LABELS output into a new sytax window, click on the VARIABLE LABELS command and click on the green arrow or "Run > Selection" to create labels for each variable.

At this point all the section data for that survey section should be successfully imported into a data window in SPSS. Hovering over a variable name should cause the label for that variable to appear.

Variable naming conventions

For questions which use checkboxes or radio buttons a variable is made for each option. The variable name is the question id followed by "__" followed by the value for that specific selection. If that variable was selected then the value for the field is "1". If the option wasn't selected the value is "0".

For any other type of input the variable name is the question id.
